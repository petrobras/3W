# How to Cite the 3W Project and Its Resources

Below are our suggestions on how to cite the 3W Project itself and/or its main resources.

We ask that all used resources be properly cited.

## 3W Project or this Git repository

```
Petróleo Brasileiro S.A. (Petrobras). The 3W Project. https://github.com/petrobras/3W (2026).
```

## 3W Dataset

### Version 1.X.X

```
Vaz Vargas, R. E., Munaro, C. J., Marques Ciarelli, P., Gonçalves Medeiros, A., Guberfain do Amaral, B., Centurion Barrionuevo, D., Dias de Araújo, J. C., Lins Ribeiro, J., & Pierezan Magalhães, L. A realistic and public dataset with rare undesirable real events in oil wells. Journal of Petroleum Science and Engineering, 181. https://doi.org/10.1016/j.petrol.2019.106223 (2019).
```

### Version 2.X.X

```
Vaz Vargas, R.E., de Melo Junior, A.J., de Campos Lima, C.B. et al. 3W Dataset 2.0.0: a realistic and public dataset with rare undesirable real events in oil wells. Sci Data 13, 949. https://doi.org/10.1038/s41597-026-07225-z (2026).
```

## 3W Toolkit

```
Petróleo Brasileiro S.A. (Petrobras). The 3W Project. https://github.com/petrobras/3W (2026).
```

NOTE: we are working on publishing an article describing the 3W Toolkit 3.X.X and this article will be specified here as soon as possible. For now, please cite any version of the 3W Toolkit in the same way that citation to the 3W Project is suggested.
